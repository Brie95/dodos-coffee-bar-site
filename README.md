# Dodo's Coffee Bar – Two‑Page Website  
A warm, cozy, fictional café website created as part of my final project for Organizational Leadership coursework. This project demonstrates my ability to plan, design, and deliver a complete web project using HTML, CSS, GitHub, and project management principles.

---

## 🌿 Project Overview  
Dodo’s Coffee Bar is a simple two‑page website designed to reflect a calm, welcoming café atmosphere. The site uses warm colors, soft textures, and friendly language to create a cozy brand identity.

This project includes:
- A **homepage** introducing the café  
- An **about page** sharing the story and values  
- A cohesive visual style using a custom CSS theme  
- A full documentation folder with scope, plan, and retrospective  
- A live GitHub Pages deployment
