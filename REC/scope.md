**Project Title:** Dodo's Coffee Bar Website

## 1. Project Purpose  
The purpose of this project is to create a simple, two‑page website for **Dodo’s Coffee Bar**, a fictional café with a warm, cozy, community‑centered vibe. This project gives me the chance to practice applying project management concepts while also building something creative and personal.  

The final website will serve as a small but meaningful example of my ability to plan, organize, and deliver a complete project from start to finish.

---

## 2. Project Description  
This project involves designing and developing a two‑page website using HTML and CSS. The site will reflect the atmosphere and personality of Dodo’s Coffee Bar — warm colors, soft textures, and a welcoming tone.  

- The **homepage** will introduce the café and highlight what makes it special.  
- The **about page** will share the café’s story, values, and overall vibe.  

The website will be published using GitHub Pages and stored in a public GitHub repository with all required documentation.

---

## 3. Desired Results  
By the end of this project, I want to deliver:

- A fully functional, two‑page website  
- A consistent visual style that matches the café’s cozy brand  
- Clear, simple navigation between pages  
- A live GitHub Pages link  
- A complete documentation folder with scope, plan, and retrospective  
- A project that feels polished, intentional, and aligned with the assignment requirements  

---

## 5. Project Exclusions  
To keep the project focused and manageable, the following are **not** included:

- JavaScript or interactive features  
- Backend systems (ordering, accounts, reservations, etc
